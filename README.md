## 🚀 Избранные проекты

> Это не туториалы — это рабочие приложения с моделями, API и интерфейсом.

### 🎥 [Detector_app](https://github.com/DenisDrobyshev/Detector_app) — Мониторинг дисциплины по видеопотоку
Веб-приложение на **Streamlit** для детекции нарушений на занятиях в реальном времени.
- **YOLO (Ultralytics)** — детекция событий на кадре (телефон, сон, еда и т.п.)
- **InsightFace** — распознавание лиц и привязка нарушений к конкретному студенту
- Работа с веб-камерой и видеофайлами (MP4 / AVI / MOV / MKV), CSV-отчёты и статистика
- `Python · Computer Vision · YOLO · InsightFace · Streamlit`

### 🗺️ [Geomarketing_analysis](https://github.com/DenisDrobyshev/Geomarketing_analysis) — Геомаркетинговая ML-аналитика
Полный аналитический пайплайн: **где открыть фитнес-центр**, чтобы максимизировать потенциал при минимуме конкуренции.
- Сбор открытых геоданных через **OpenStreetMap / OSMnx**, гексагональная сетка **H3**
- Обучение модели привлекательности локаций (**scikit-learn**) + интерактивные карты **Leafmap**
- От сырых геоданных — к обоснованному бизнес-решению
- `Python · Geospatial ML · H3 · scikit-learn · OSM`

### 🐾 [classification_animals](https://github.com/DenisDrobyshev/classification_animals) — Классификатор изображений
Сравнение архитектур (Dense / SimpleCNN / VGG-like) на 5 классах животных, с деплоем.
- Лучшая модель **VGGCNN — F1 0.88** (Dense 0.70 → SimpleCNN 0.87 → VGGCNN 0.88)
- Деплой: **Streamlit** UI + **FastAPI** инференс-API
- `Python · TensorFlow / Keras · CNN · Streamlit · FastAPI`

### 🗃️ [excel-to-db-converter](https://github.com/DenisDrobyshev/excel-to-db-converter) — Excel → база данных
Бэкенд-приложение для превращения Excel-файлов в структурированные таблицы БД.
- Авторизация, настраиваемые шаблоны таблиц, валидация данных по типам и правилам
- Веб-интерфейс для просмотра и редактирования записей, совместная работа над данными
- `Python · FastAPI · Uvicorn · HTML / JS`

### 🐉 [chinese-premium-store](https://github.com/DenisDrobyshev/chinese-premium-store) — E-commerce на React
Интернет-бутик с каталогом, корзиной и избранным.
- Фильтры / сортировка / пагинация, карточка товара с галереей, «недавно просмотренные»
- **React Router**, **Context API** для корзины, хранение wishlist в LocalStorage
- `React · React Router · Context API · CSS`

### 🛩️ [StableDroneSystemAnalysis](https://github.com/DenisDrobyshev/StableDroneSystemAnalysis) — Анализ устойчивости БПЛА
Инженерный инструмент моделирования динамики и устойчивости систем управления дрона.
- Интегрирование уравнений движения (**Эйлер**, **Рунге-Кутта 4-го порядка**), анализ собственных чисел
- Фазовые портреты, сравнение стратегий управления, интерактивный GUI
- `Python · NumPy · Control Theory · Tkinter`
